Survey Widget Demo

This is a demo page to test the survey popup widget. After 3 seconds, a popup should appear in the bottom-right corner of the page.

The popup is configured via the /survey-config endpoint on the server.

How it works

  1. The embed.js script is loaded via a script tag
  2. It dynamically loads widget.js which contains the popup logic
  3. The widget fetches configuration from the server
  4. After the configured delay, the popup appears
